# GAMUT OF BLOB
A puzzle game created by David 'dmatter' Gill for "The Week of Awesome IV" Gamedev.net Competition (Aug 8th - Aug 14th 2016).
**Won 1st place!**



## CONTROLS
Movement - WASD or ARROW KEYS
Switch Blob - SPACE
Reset Level - R
Skip level - EQUALS (=)
